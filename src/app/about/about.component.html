A Simple Binary Machine

Charles Petzold's book Code: The Hidden Language of Computer Hardware and Software is a fascinating read. My favorite part is chapter 12: A Binary Adding Machine. The chapter explains how one could make a machine able to add two 8-bit binary numbers together using 144 relays. As I learn best by doing (and lack soldering skills) I decided to make the virtual one below using the logic Mr. Petzold provided in the book.
All the switches on the top row are for the first number to add, and all the switches below are for the second number. Click the switches to toggle them between 0 and 1 for that bit.
For example 0110-0101 + 1011-0110 is top row: (down, up, up, down - down, up, down, up) second row: (up, down, up, up - down, up, up, down). The sum of the operation is 1-0001-1011 which is shown on the bulbs as: (on - off, off, off, on - on, off, on, on). This tool is best used in a desktop browser.
